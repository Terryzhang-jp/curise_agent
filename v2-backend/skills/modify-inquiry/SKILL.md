# 修改询价单

当用户要求修改已生成的询价单时，按以下步骤操作：

## 步骤 1：确认订单和文件

调用 `get_order_overview(order_id)` 确认订单状态和已生成的询价文件列表。
如果用户没指定 order_id，问清楚。

## 步骤 2：定位文件

从 overview 的"询价单"部分找到对应供应商的文件名。
调用 `modify_excel(filename="<文件名>", action="list")` 确认文件在 workspace 中存在。
如果不存在，提示用户可能需要重新生成。

## 步骤 3：读取当前内容

调用 `modify_excel(filename="<文件名>", action="read", cells="A1:J30")` 查看当前内容。
根据用户要求定位要修改的单元格。

## 步骤 4：执行修改

调用 `modify_excel(filename="<文件名>", action="write", cells='{"H8": "2026/04/15"}')` 写入新值。
cells 参数是 JSON 格式：`{"单元格": "新值"}`。

多个修改可以一次完成：`{"H8": "2026/04/15", "I22": "500", "J22": "=I22*K22"}`

## 步骤 5：确认结果

修改完成后告知用户文件已更新，可以下载查看。
如果用户还要改别的，回到步骤 3。

## 注意事项

- 修改后的文件保存为 `<原名>_modified.xlsx`，不覆盖原始文件
- 数字值直接写数字（如 `500`），不要加引号
- 公式以 `=` 开头（如 `=I22*K22`）
- 日期格式统一用 `YYYY/MM/DD`（如 `2026/04/15`）
